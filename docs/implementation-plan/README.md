# Implementation Plans

This directory is the index for implementation plan packs.

## Structure

Each concrete plan lives in its own slugged subfolder:

```text
docs/implementation-plan/
├── README.md
└── <plan-slug>/
    ├── README.md
    ├── 01-requirements.md
    ├── 02-design.md
    └── 03-tasks.md
```

## Naming

- Use lowercase letters, digits, and hyphens only
- Prefer short, stable slugs
- Create a new slugged folder for a new plan instead of overwriting another plan pack
